
Node.js+Express+MySQL学生选课系统
===

>
> 	微型的学生选课系统，提供对数据库最基础的增删改查功能。
>
>* 面向学生功能：
> 
>     1. 查看所有课程，并进行选课
> 
>     2. 查看所有课程，并进行退课
> 
>     3. 查看个人信息，并进行修改
> 
>     4. 查看已选课程的任课老师信息
>
>*  面向教师功能：
> 
>     1. 查看所有课程，并进行任课
> 
>     2. 查看已带课程，并进行退课
>

## 版本号

>
> **Express** v4.14.0
